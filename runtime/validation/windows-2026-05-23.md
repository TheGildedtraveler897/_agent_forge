# Windows Manual Smoke Checklist

Date prepared: 2026-05-23

Run this on the fresh Windows VM used for the ship smoke test.

1. Transfer these files to the same folder on the VM:
   - `agent-forge-suitcase-20260523-validation.zip`
   - `agent-forge-suitcase-20260523-validation-deploy-and-bootstrap.ps1`
2. Open Windows PowerShell 5.1 or newer in that folder.
3. Run:

   ```powershell
   powershell.exe -ExecutionPolicy Bypass -File .\agent-forge-suitcase-20260523-validation-deploy-and-bootstrap.ps1 -BundleZip .\agent-forge-suitcase-20260523-validation.zip -DestinationRoot .\af
   ```

4. Confirm extraction completed and the deployed tree contains:
   - `af\agent-forge-suitcase-20260523-validation\_agent_forge\AGENTS.md`
   - `af\agent-forge-suitcase-20260523-validation\_agent_forge\docs\`
   - `af\agent-forge-suitcase-20260523-validation\_agent_forge\policies\`
   - `af\agent-forge-suitcase-20260523-validation\_agent_forge\scripts\deploy-factory.ps1`
5. Run:

   ```powershell
   cd "$HOME\Projects\_agent_forge"
   py -3 scripts\verify-agent-forge.py
   ```

6. Open Claude Code and run `/onboarding-guide`.
7. Acceptance gate: Beat 0 renders inline in chat, not only as a subprocess or terminal output.
8. Capture screenshots and command output under `runtime\validation\windows-2026-05-23\` on the source repo before tagging the release.
